Gerber Files Explained: How to Export, Send & What Manufacturers Need

Published June 10, 2026 · By Mayio Engineering Team · 7 min read

Gerber files are the universal language of PCB manufacturing. Every factory in the world — from Shenzhen to Germany to Ohio — reads the same format. If you can export Gerbers, you can get your board made anywhere.

What Are Gerber Files?

Gerber files describe each layer of your PCB as a separate image. Think of them like X-ray slices — one file for the top copper, one for bottom copper, one for silkscreen, one for solder mask, and so on.

How to Export from Popular EDA Tools

KiCad

  1. Open your PCB layout
  2. File → Fabrication Outputs → Gerbers (.gbr)
  3. Select all layers you need
  4. Then: File → Fabrication Outputs → Drill Files (.drl)
  5. Zip everything into one file

Eagle

  1. File → CAM Processor
  2. Select "gerber" from the job dropdown
  3. Process Job → save all files
  4. Zip into one file

Altium Designer

  1. File → Fabrication Outputs → Gerber Files
  2. Set units and format (mm, 4:4 is standard)
  3. Select all layers
  4. Run → then File → Fabrication Outputs → NC Drill Files
  5. Zip everything

EasyEDA

  1. File → PCB Fabrication File (Gerber)
  2. Download the zip directly
  3. Ready to send

Common Mistakes

1. Sending Screenshots Instead of Gerbers

A screenshot of your PCB layout in your EDA tool is not a Gerber file. The factory cannot manufacture from a screenshot. Always export properly.

2. Missing Drill File Units

If your Gerber is in mm but your drill file is in mils, the holes will be 25.4× too far apart (or too close). Always check: File → Drill Setup → Units should match your Gerber units.

3. No Board Outline

If the factory can't see where to cut, they'll ask. This adds a day. Make sure your Edge.Cuts layer is exported.

4. Forgetting Inner Layers (4+ Layer)

For 4-layer boards, you need inner copper layers (.gl2, .gl3). Missing these means the factory has to guess your stackup.

5. Not Zipping Everything

Send one zip file with all Gerbers + drill files. Not individual files. Not a RAR. One .zip.

Pro tip: Before sending to the manufacturer, open your Gerber zip in a free viewer like [Gerber Viewer](https://www.gerber-viewer.com/) or the Ucamco reference viewer. Visually confirm all layers are present and look correct.

What Your Manufacturer Does With Them

  1. Import & Parse — Load all Gerber layers into their CAM software
  2. DFM Check — Run design for manufacturability checks
  3. Panelization — Arrange boards on a manufacturing panel for efficiency
  4. Generate Phototooling — Create the masks used to etch copper
  5. Quote — Calculate pricing based on layer count, size, quantity, and specs
